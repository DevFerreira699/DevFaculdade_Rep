Para criar um novo repositorio

git init
git add README.md
git commit -m "first commit"
git branch -M main
git remote add origin git@github.com:DevFerreira699/0.git
git push -u origin main

para adicionar em um repositorio já existente

git remote add origin git@github.com:DevFerreira699/0.git
git branch -M main
git push -u origin main
